mb_Clipboard-Search Plugin

Description:

Allows you to assign a hotkey to search your Library using the text in your Windows Clipboard.


Instructions: 

1. Drag mb_Clipboard-Search.dll from the Plugins folder to your MusicBee Plugins folder, typically located at: C:\Program Files (x86)\MusicBee\Plugins.
2. Open MusicBee and go to Edit > Edit Preferences > Plugins and enable mb_Clipboard-Search.
3. Go to the Hotkeys section of the Preferences Pane and scroll down to "General: Search for Clipboard Text". Assign a hotkey.
